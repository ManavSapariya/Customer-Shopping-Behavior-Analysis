# 📌 Project Overview

This project analyzes customer shopping behavior using transactional data from **3,900 purchases** across multiple product categories. The objective is to discover meaningful insights into customer demographics, purchasing habits, subscription behavior, and revenue trends to support data-driven business decisions.

The project follows a complete analytics workflow:

- Data Cleaning & Preparation using Python
- Data Storage & Analysis using MySQL
- Business Querying using SQL
- Interactive Dashboard Development using Power BI
- Business Recommendations based on analytical findings

# 📊 Key Business Insights

### Customer Overview

- Total Customers: **3,900**
- Average Purchase Amount: **$59.76**
- Average Review Rating: **3.75**

### Subscription Insights

- 27% Customers are Subscribers
- 73% Customers are Non-Subscribers

### Revenue Insights

- Clothing generates the highest revenue.
- Accessories is the second highest performing category.

### Customer Segments

- Young Adults contribute the highest revenue.
- Loyal customers dominate the customer base.
